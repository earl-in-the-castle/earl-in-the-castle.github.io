'教学任务-作业 作品集'

教学任务-作业 作品集

  • 请求路径前缀

https://test2.fifedu.com/iplat/

  • 请求路径

/bp/teacher/home/teacherTaskList

  • 请求参数

无

  • 说明

作品集 https://test2.fifedu.com/ifass/teacher/record/subCollectionList?taskCollectionId=3b48772dc12443a6bb9bbf1ce725d7cf&classId=2811000226000025852

作业 https://test2.fifedu.com/ifass/teacher/task1/taskDetails?tid=c14989492a1849a9aead6312661614cf

  • 返回参数
参数 类型 允许为空 说明
status String NO 接口操作状态：SUCCESS表示操作成功，ERROR表示操作异常（会在message中说明异常原因）
message String NO status的辅助字段，用于对status辅助说明
data Array YES 接口返回数据
  taskAndCollectionList list NO 作业和作品集列表
    flg String NO 作业或者作品集标识（1 作业/2 作品集）
    courseId String NO 课程id
    courseName String NO 课程名称
    className String YES 班级名称
    classId String YES 班级id
    taskName String NO 作品集名称或者作业名称
    taskId String NO 作品集名称或者作业id
    memberId String YES 学生id
    userName String YES 学生名字
  taskLink String NO 作业访问链接 后面拼接参数按照说明格式 （tid对应taskId）
  collectionLink String NO 作品集访问链接 后面拼接参数按照说明格式 （taskCollectionId对应taskId ，classId对应classId）
  • 说明
    作品集只显示的是学生姓名
    作业只显示的是班级信息
